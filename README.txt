TP 4: Proyecto Final | Generar figura con la transformada de Fourier
Integrantes: Bartolomeo Nicolás, Baxendale Julian, González Matías y Dinardo Bruno
Trabajo realizado con Python + Framework Streamlit.

Versión de Python recomendada: Conda 3.11.5

Dependencias utilizadas:
streamlith: conda install -c conda-forge streamlit
streamlit-drawable-canvas: pip install streamlit-drawable-canvas
numpy: conda install -c anaconda numpy
matplotlib: conda install -c conda-forge matplotlib
pandas: conda install -c anaconda pandas

*Se recomienda realizar estas instalaciones en un entorno de Anaconda*
